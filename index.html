REACT

리엑트를 사용한 프론트 뷰 및 REST API를 사용해 백엔드와의 서버 연결 경험이 있습니다. JSX를 사용하여 탬플릿 제작을 했으며, API 활용(공공데이터포털, Naver, 서울데이터광장 등)하여 서버의 데이터를 받아 구현한 경험이 있습니다.
